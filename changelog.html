Changelog

We're constantly shipping. Here is a rundown of the latest updates and improvements to Polymer.

January 30, 2024

Select candidates for bulk actions

We improved bulk actions for candidates by adding ability to select or deselect candidates in a hiring stage before sending a bulk message, or bulk moving candidates to a different stage.

Candidate uploads

A new custom question type has been added that allows candidates to upload files to their applications. Supported file types include audio files, image files, excel/csv, word/pdf (document) and txt files.

Other improvements & fixes

Added list-unsubscribed header to all emails to improve deliverability

An issue where some incoming candidate messages were parsed incorrectly is fixed

December 20, 2023

Review Request

Review Requests allow hiring team members to request candidate reviews from any member of the organization with any user role. This will send an email to the organization email with a link directly to the review page for that candidate!

Any hiring team member can request a review via email address from someone who is not yet a member of the organization. By clicking on the review link in the email they will be added to the organization as an interviewer.

Interviewer Role

The interviewer role is geared towards individuals who don’t need or want to be added to a job’s hiring team, but will interview a candidate. This role allows individual candidate review requests to be sent to “interviewers” which will give them access to that candidates review kit page and nothing else. This allows them to easily submit reviews.

Other improvements & fixes

Hiring Team Members are able to delete custom questions no matter their user role

If the owner of an organization update’s their information, it will now be properly updated for Stripe/Billing (shows on the invoices and receipts)
